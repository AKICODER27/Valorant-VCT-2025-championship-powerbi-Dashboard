🎮 Valorant Champions 2025 Tournament Analytics Dashboard🎮

📌 Project Overview

This project is an interactive Power BI Dashboard built using Valorant Champions 2025 tournament statistics. The dashboard provides comprehensive insights into player performance, team performance, and agent analytics through interactive visualizations, KPIs, and filters.

The goal of this project is to demonstrate practical Data Analytics, Business Intelligence, Data Visualization, and Power BI skills by transforming raw esports data into meaningful insights for decision-making.

💡 Key Insights
MIBR emerged as one of the highest-rated teams in the tournament.

DRX recorded the highest total kills among all teams.

aspas achieved the highest average player rating.

Flashback secured the highest total kills among players.

Omen was the most frequently selected agent.

Yoru delivered the highest Average ACS.

Higher ACS generally correlated with higher player ratings.

Teams with stronger ACS and kill counts consistently performed better.
